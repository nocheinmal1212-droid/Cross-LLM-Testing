## Description
The models tested in this experiment are: Qwen-7B-Chat, ChatGLM3-6B and Baichuan2-7B-Base. The model files are absent. 
Once you have them, run 

python3 test_three_models_v2.py 

to test.
The two programs, run_glm and test_three_models, are preserved as part of the experimental process. 
The logs contain the conversation with each individual model and a combined one.
